Prework Study Guide

HTML

  • The head element contains metadata that helps the browswer render the page correctly.
  • The body element represents the visible content shown to the user.
  • An HTML attribute defines additional information about the element.
  • Metadata contains information for the browser about accessibility, search engines and performance.
  • The Title element defines the name of the page.
  • The style element defines the CSS style.
  • The base element defines the base URL for the page.
  • The link element connects an external resource to the HTML document.
  • The script tag embeds client-side scripts into the HTML document.
  • Elements nested inside other elements are refered to as child elements.
  • Elements with other elements nested inside them are refered to as parent elements.
  • Semantic HTML elements are those whose name describes the element.
  • The Footer is located at the bottom of the page and often contains info on the author, copyright, contact information, sitemap and navigation.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • There are 3 ways to use CSS to stylize a website: inline CSS, internal CSS and external CSS.
  • A selector defines the element to which the declaration will apply.
  • Declarations should be separated by semi-colons. Multiple can be used after one selector.
  • Declarations contain 2 components: The CSS property we want to apply and the value of the property.
  • A class selector is a special wildcard symbol used to apply to a group of elements.
  • A class selector is assigned in HTML.
